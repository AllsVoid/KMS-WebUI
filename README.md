## ⚙️ 构建

```
自行构建前需要拥有 node.js, pnpm 等必要依赖
```

### 克隆代码

```bash
git clone https://github.com/ikxin/kms-tools.git
```

### 构建打包

```bash
# 进入项目
cd kms-tools
# 安装依赖
pnpm install
# 开发环境
pnpm run dev
# 开始构建
pnpm run build
# 预览构建
pnpm run preview
```
